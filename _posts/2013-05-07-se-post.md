# 安装git：
1、通过下载地址：http://pan.baidu.com/share/link?shareid=417696&uk=607566152，
   下载railsinstaller-2.1.0.exe，一键安装 
安装github：
2、改淘宝源，在cmd模式下依次输入
    gem sources --remove http://rubygems.org/
    gem sources -a http://ruby.taobao.org/
    gem sources -l
3、点击安装GitHubSetup.exe
